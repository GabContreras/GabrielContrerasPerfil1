Componentes Activos:

Listado de componentes

Switch

Dispositivo que conecta múltiples dispositivos dentro de una red, gestionando el tráfico de datos.

Funcionalidad: Puede operar en diferentes capas del modelo OSI, siendo fundamental en la segmentación y gestión del tráfico de red.

Router

Dispositivo que conecta redes diferentes y dirige el tráfico de datos entre ellas. También puede asignar direcciones IP y realizar funciones de seguridad como firewall.

Aplicaciones: Es esencial para conectar redes domésticas o de oficina a Internet.

Tarjeta de red (NIC)

La Tarjeta de Red, también conocida como Network Interface Card (NIC), es un componente crucial que permite a un dispositivo, como una computadora, conectarse a una red, ya sea mediante un cable físico o de forma inalámbrica. La NIC actúa como una interfaz entre el dispositivo y la red, permitiendo la transmisión y recepción de datos.

Interruptor inteligente Wifi

Un Interruptor Inteligente WiFi es un dispositivo domótico que permite el control remoto de dispositivos eléctricos a través de una red inalámbrica. Estos interruptores se integran con sistemas de automatización del hogar y pueden ser gestionados mediante aplicaciones móviles o asistentes de voz.

Access Point

Un dispositivo que permite a los dispositivos inalámbricos conectarse a una red cableada. Funciona como una puerta de entrada entre la red inalámbrica y la red física.

Características: Puede soportar múltiples estándares de Wi-Fi (802.11a/b/g/n/ac) y ofrece funciones avanzadas como la gestión de usuarios y la seguridad.



Herramientas utilizadas para realizar cableado estructurado:

Listado de herramientas

Ponchadora

Herramienta para insertar y asegurar cables en jacks o paneles.

Es esencial para la instalación de cables en los conectores y paneles de conexión. Esta herramienta permite insertar los cables en los conectores, cortando el exceso y asegurando una conexión firme y fiable.

Crimpadora

Herramienta para fijar conectores RJ45 en los extremos de un cable UTP.

Este proceso es esencial para crear cables de red personalizados con las longitudes y configuraciones necesarias para instalaciones específicas.

Desforradora

Herramienta para quitar el recubrimiento de cables sin dañar los hilos internos.

Esto es fundamental para preparar los cables antes de la crimpación o conexión en paneles.

Tester/ probador de cables

Dispositivo para verificar la correcta conexión y funcionalidad de los cables de red.

Este dispositivo es crucial para asegurar que las conexiones se han realizado correctamente y que no hay cortocircuitos, cruces de cables o conexiones abiertas.

¿Qué es el cableado horizontal?

El cableado horizontal es la infraestructura que conecta las estaciones de trabajo (como computadoras, teléfonos, y otros dispositivos de red) con el cuarto de telecomunicaciones dentro de la misma planta de un edificio. Es una parte esencial del sistema de cableado estructurado en cualquier instalación de red.

Es utilizado para conectar dispositivos como computadoras, impresoras, teléfonos VoIP, y otros equipos de red a la infraestructura principal de telecomunicaciones.
