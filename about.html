Componentes Activos:

Listado de componentes

Switch

Dispositivo que conecta múltiples dispositivos dentro de una red, gestionando el tráfico de datos.

Router

Dispositivo que dirige el tráfico entre diferentes redes y proporciona acceso a internet.

Tarjeta de red (NIC)

Componente que permite la conexión de un dispositivo a la red.

Interruptor inteligente Wifi

Dispositivo que permite controlar dispositivos eléctricos a través de una red inalámbrica.

Access Point

Dispositivo que extiende la cobertura de una red inalámbrica.



Herramientas utilizadas para realizar cableado estructurado:

Listado de herramientas

Ponchadora

Herramienta para insertar y asegurar cables en jacks o paneles.

Crimpadora

Herramienta para fijar conectores RJ45 en los extremos de un cable UTP.

Desforradora

Herramienta para quitar el recubrimiento de cables sin dañar los hilos internos.

Tester/ probador de cables

Dispositivo para verificar la correcta conexión y funcionalidad de los cables de red.

¿Qué es el cableado horizontal?

El sistema de cableado horizontal es la porción del sistema de cableado de telecomunicaciones que se extiende del área de trabajo al cuarto de telecomunicaciones o viceversa.

Es la infraestructura de cableado que conecta los dispositivos en una misma planta de un edificio.
