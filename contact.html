¿Para qué se utiliza un cable directo en la red?

Un cable directo, también conocido como cable de par trenzado directo, es un tipo de cable Ethernet en el cual los pines de los conectores RJ45 en ambos extremos están conectados en un orden idéntico. Este tipo de cable es ampliamente utilizado para la conexión de dispositivos diferentes dentro de una red.

El cable directo se utiliza para conectar diferentes tipos de dispositivos entre sí, como un ordenador a un switch, un router, o un módem. Es el cable estándar para la mayoría de las conexiones de red dentro de una red doméstica o de oficina.

¿Para qué se utiliza un cable cruzado en la red?

Un cable cruzado es un tipo especial de cable Ethernet en el cual los pines de los conectores RJ45 en un extremo están conectados en un orden diferente al del otro extremo. Este tipo de cable es utilizado principalmente para conectar dispositivos del mismo tipo directamente entre sí.

Un cable cruzado es ideal para conectar dos dispositivos similares, como dos ordenadores, dos switches, o dos hubs, sin necesidad de un switch o un router intermedio.
